# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-XX

### Added
- Initial release
- Job Shop Gymnasium environment
- Heuristic agents (SPT, LPT, FIFO, MWR, Random)
- Q-Learning simple (table-based)
- Deep Q-Network with PyTorch
- PPO simple implementation
- Benchmark instances (FT06, FT10, FT20, LA01, LA02)
- Gantt chart visualization
- Training curves plotting
- Complete documentation

### Results
- Deep DQN achieves 69 makespan on FT06 (25.45% gap from optimal)
- 36.7% improvement over SPT heuristic
